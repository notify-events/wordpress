Notify.Events plugin is ultimate tool for any kind of notifications from your WordPress website and to more then 20 messengers and platforms such as SMS, voicecall, Facebook messenger, Viber, Telegram, VK, Skype, Slack and etc (down below full list of implemented recepients)

== Description ==

Plugin allows you to receive notifications from Wordpress about new comments, posts, users registrations. Also it provides a way to get notifications from WooCommerce, Contact Form 7, WPForms with additional plugins

== Installation ==

1. Install plugin Notify.Events Ultimate Notifications
3. Activate plugin through the **Plugins** screen (**Plugins > Installed Plugins**)
4. Sign up on website [Notify.events](https://notify.events)
5. Create and name a channel
6. Add source "WordPress" and acquire token. Follow the instructions
7. Subscribe in your channel to platform you choose
8. Go to your website plugin settings. Choose tab channel. Name a channel and paste the token.
9. Go to tab Events. Add an event. Done!
